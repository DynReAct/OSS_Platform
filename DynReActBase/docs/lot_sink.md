# Lot sink

The LotSink module defines the LotSink interface, allowing to transfer lots generated by the DynReAct 
mid-term planning algorithm to an external data sink.
While a custom implementation is indispensable for productive use, a file-based implementation for test and dev 
purposes is included in the open-source package. It can be enabled by means of the environment variable:

```
LOT_SINKS=default+file:./testlots
```

Where *./testlots* is the folder to store lots in.
