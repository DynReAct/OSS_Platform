Agent class

Class Generic Agent. supporting the basic concepts.

Attributes: topic (str): Topic driving the relevant converstaion. agent (str): Name of the agent creating the object. kafka_ip (str): IP address and TCP port of the broker. verbose (int): Level of details being saved.
