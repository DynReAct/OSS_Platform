Material class

Bases: Agent

Class Material supporting the material agents.

Arguments: topic (str): Topic driving the relevant converstaion. agent (str): Name of the agent creating the object. params (dict): parameters relevant to the configuration of the agent. kafka_ip (str): IP address and TCP port of the broker. verbose (int): Level of details being saved.

handle_create_action(dctmsg)

Handles the CREATE action. It clones the master MATERIAL for one auction. This instruction should only be given to the general MATERIAL.
